## 三、API 接口文档

### 3.1 背景移除 API

**端点**: `POST /api/remove-bg`

**请求**:
```
FormData:
  image_file: File (JPG/PNG/WEBP, max 10MB)
```

**响应** (成功):
```
Headers:
  Content-Type: image/png
  X-Credits-Remaining: number
  X-Is-Guest: "true" | "false"

Body: PNG 二进制数据
```

**错误响应**:
```json
{
  "error": "错误信息",
  "code": "ERROR_CODE"
}
```

**响应** (成功):

## 八、常见问题排查

### 8.1 AI Editor 生图失败

**检查项**:
1. 环境变量 `REPLICATE_API_TOKEN` 是否配置
2. Replicate 账户余额是否充足
3. 模型版本 ID 是否正确（当前使用 `black-forest-labs/flux-schnell`）

### 8.2 背景移除失败

**检查项**:
1. 环境变量 `REMOVE_BG_API_KEY` 是否配置
2. remove.bg 账户余额是否充足
3. 图片格式/大小是否合规

### 8.3 积分扣除异常

**检查项**:
1. `profiles` 表 RLS 策略是否允许读写
2. `deduct_credits` 函数执行是否正常
3. 外键约束 `profiles_id_fkey` 是否已删除（解决 NextAuth 兼容性问题）
